Online Room Booking Management System (University)

This was my BSc final year development project. It was the first time developing a web application and I used: HTML, CSS, JavaScript, Node.js, Express.js,and MySQL/SQL, The idea was to create an application from which one can book rooms. Additionally the application would analyse the bookings and try to block certain rooms from being bookable to try and space out crowds. Ideally this would reduce the spread of covid and alikes. Alas this feature is yet to be implemented, it is on my list for future projects.
